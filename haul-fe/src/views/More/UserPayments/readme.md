## 페이지 설명
사용자의 결제 수단 정보를 확인할 수 있는 페이지입니다.<br />
캐로셀로 소유한 카드를 넘기며 확인할 수 있습니다.

헤더의 뒤로가기 버튼을 이용하여 다른 페이지로 넘어갈 수 있습니다.

## 기능 설명
1. 캐로셀을 넘겨 카드를 선택할 수 있습니다.
2. 결제수단 추가하기 버튼을 누르면 안내 토스트가 출력됩니다.
3. 헤더의 뒤로가기 버튼으로 이전 페이지로 돌아갈 수 있습니다.
