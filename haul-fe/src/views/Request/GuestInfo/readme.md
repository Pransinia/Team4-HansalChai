## 페이지 설명
비회원으로 이용 시 필요한 정보를 입력하는 화면입니다.<br/>
로그인을 하지 않은 사용자만 운송 품목 정보 입력 후 접근할 수 있습니다.

이름과 전화번호를 입력합니다.

## 기능 설명
1. 이름을 입력받습니다. 이름은 빈 문자열인지 검사합니다.
2. 전화번호를 입력받습니다. 입력된 값이 전화번호 양식을 따르는지 감사합니다.
3. 이름과 전화번호가 모두 입력되었고, 두 정보 모두 검사를 통과했다면 다음 단계로 진행할 수 있습니다.
